BindingNavigator 控制項範例

[This readme is a partial copy of the sample's topic in the SDK documentation.]

這個範例會向您示範如何使用與 BindingSource 元件繫結的 BindingNavigator 控制項。

BindingNavigator 控制項會從 ToolStrip 衍生，同時包含預先填入的一組按鈕，可供巡覽 BindingSource 元件中的資料。而且，這些按鈕的功能都是可以擴充的。在這個範例中，已經覆寫了 [New] 按鈕的行為。

如需使用這些範例的詳細資訊，請參閱下列主題：

  • HOW TO：檢視和下載範例

  • HOW TO：設定範例設定

安全性注意事項：

這個程式碼範例僅供概念說明之用，同時因為範例中的程式碼撰寫並沒有周全的顧及所有安全性問題，所以本範例不適用於實際執行的應用程式或網站中。如果使用者擅自將範例程式碼用於其他意圖而導致連帶事件或重大損害時，Microsoft 將不承擔任何責任。

若要使用命令提示字元建置及執行範例

  1. 在命令提示字元中，巡覽至您用來儲存未壓縮的範例檔案的位置。。

  2. 巡覽到 BindingNavigatorSample 目錄下任一程式設計語言的子目錄。

  3. 輸入 msbuild，使用提供的方案 (.sln) 檔來建置範例。

  4. 輸入 cd BindingNavigatorSample\bin\Debug，巡覽到可執行檔的所在位置。

  5. 輸入 BindingNavigatorSample.exe 即可執行範例。

若要使用 Visual Studio 建置和執行這個範例

  1. 在 [Windows 檔案總管] 中，巡覽至您用來儲存未壓縮的範例檔案的位置。

  2. 巡覽到 BindingNavigatorSample 目錄下任一程式設計語言的子目錄。

  3. 按兩下方案 (.sln) 檔，在 Visual Studio 中開啟該方案檔。

  4. 在 Visual Studio 中，按 F5 進行建置並執行範例。

若要使用 BindingNavigatorSample

  • 從 ComboBox 控制項選取一個旗標，然後按一下加號 (+) 按鈕，建立 Flag 物件，並將它加入至 BindingSource 元件。如果想要移動已加入的 Flag 項目，可以使用 BindingNavigator 控制項上的箭號按鈕。

示範

  • 使用 BindingNavigator 控制項，移動資料來源中的所有項目

  • 使用 BindingSource 元件，將 Windows Forms 控制項繫結至資料來源
